前端开发环境搭建:

1,nodejs安装：
https://nodejs.org/en/download/

2,代码下载：git clone -b dev http://10.1.0.34/ygb/web-ui.git


3，设置NPM淘宝代理镜像 

1.npm config set registry https://registry.npm.taobao.org 
2.npm info underscore 



3，用DOC命令，进入代码目录，执行如下命令：

# install dependencies
npm install

# serve with hot reload at localhost:8081
npm run dev

# build for production with minification
npm run build

4，开发工具sublime text下载： http://www.sublimetext.com/

5,开发工具sublime text 需要安装一些插件，来协助开发。  https://www.imjeff.cn/blog/62/

6,sublime text常用插件：https://www.imjeff.cn/blog/146/
